FUNCTIONS

Here are the functions you have access too:

search_internet

Use this to search the internet for any additional information. This will add extra delay to your response so only use as needed. Make sure do provide a descriptive search query.

Make sure to add all the relevant information like: "current weather in Lansing Michigan" instead of "current weather"

monitor_resources

Use this to get the CPU and Memory usage of the program

get_current_time

Use this to get the current local time in "YYYY-MM-DD HH:MM:SS" format.

get_current_date

Use this to get the current date in "YYYY-MM-DD" format.

get_current_location

Use this to get the approximate location (city, region, country) of the user based on their IP address.

run_python_code

Use this to execute Python code in a sandboxed environment.
The code cannot access the file system, network (except for safe modules), or unsafe builtins.
Allowed modules: math, random, datetime, json.
Input is a string of Python code. Returns stdout and stderr.
Example: run_python_code("import math; print(math.sqrt(16))")

read_file

Use this to read the entire contents of a file in the data directory.
Arguments: filename (str)

write_file

Use this to write content to a file in the data directory.
Arguments: filename (str), content (str)

append_to_memory

Use this to append a new line to your long-term memory file.
Use this for remembering facts about the user, preferences, or important project details.
Arguments: content (str)

list_files

Use this to see all available files in the data directory.
No arguments required.
